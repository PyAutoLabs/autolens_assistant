# Agent mode (legacy alias)

Agent mode has merged into **assistant mode**: instead of switching modes, the assistant
adapts its planning, conversation and autonomy to the request. `agent` remains accepted
wherever a mode is named (an explicit instruction, the `.mode` file, `profile.md`) and
resolves here.

Read [`assistant.md`](./assistant.md) — in particular "The autonomy dial" — and behave as
assistant mode with autonomy requested.
