# SLACS0946+1006 subhalo pipeline

## Domain motivation

SLACS0946+1006 is a target for resolved-arc gravitational imaging of dark-matter
substructure. The analysis is framed as the standard [[substructure_and_subhalos]]
problem: fit a smooth lens model, then ask whether a compact perturber improves
the model enough to overcome the Bayesian evidence penalty for additional
parameters.

## Statistical motivation

The detection statistic is `Delta ln Z`, comparing the evidence of the smooth
model against the evidence of the model with a subhalo. The SIS search maps
`Delta ln Z` across the image plane, then a local refinement gives the detection
evidence used for the final model comparison. The concentration test compares
the refined SIS evidence against an NFW perturber at the same recovered position.
This follows the grid-search workflow described in [[substructure_and_subhalos]]
and uses a pixelized source reconstruction as described in
[[inversions_and_pixelizations]].

## Implementation choice

The script `scripts/imaging.py` runs a SLaM-style sequence: MGE lens and source
initialisation, a pixelized source phase, a refined lens-light phase, a power-law
smooth mass baseline, an SIS subhalo grid search, an SIS refinement and an NFW
comparison at the SIS position. Outputs are written under
`output/imaging/slacs0946+1006/slacs0946_subhalo/`, and the final evidence
summary is written to `work/output/slacs0946_subhalo/evidence_summary.json`.

The analysis is not laptop-fast: the default six-by-six SIS grid requires dozens
of nested-sampling fits after the SLaM baseline. The imaging SLURM submit scripts
therefore point at `dataset/imaging/slacs0946+1006/` for HPC execution.
