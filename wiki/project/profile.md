# Project profile

## Lensing background

The user is comfortable enough with strong-lens substructure literature to ask for
Bayesian model comparison on SLACS0946+1006 and for a test of the compact,
"super-concentrated" subhalo interpretation.

## PyAutoLens background

_unrecorded_

## Current science goal

Reanalyse HST imaging of SLACS0946+1006 to reproduce the published dark-matter
subhalo detection, comparing a smooth-mass baseline against a free-position SIS
perturber and then comparing the recovered SIS perturber against an NFW profile
at the same position.

## Data on hand

One HST imaging dataset in `dataset/imaging/slacs0946+1006/`, with data,
noise-map, PSF, extra-galaxy mask, image positions and metadata in `info.json`.
