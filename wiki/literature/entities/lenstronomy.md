# lenstronomy

## What it is

The other major open-source strong-lens modelling code (Birrer & Amara
2018). Widely used for [[time-delay-cosmography|TDCOSMO/H0liCOW]]
analyses; comparable scientific scope to [[pyautolens|PyAutoLens]] but
different in architecture and API. PyAutoLens users will often see
lenstronomy cited as the reference implementation for a given technique.
