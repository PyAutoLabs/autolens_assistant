# Sources: degeneracies and systematics

## Wertz 2018 — pySPT

**File:** `Strong_Lens/Wertz2018PySPT.pdf`
**Concepts:** [[source-position-transformation]],
[[mass-sheet-degeneracy]]
**Summary (stub):** Public Python tool for computing
Source-Position-Transformation-equivalent lens models. Lets users assess
how much of an inferred model is degenerate.

## Kochanek 2020 — radial-angular

**File:** `Strong_Lens/Kochanek2020radialangular.pdf`
**Concepts:** [[radial-angular-degeneracy]]
**Summary (stub):** Argues lensing data fundamentally constrain the
angular structure of the potential much better than the radial — only
the mass at θ_E is genuinely tight.
