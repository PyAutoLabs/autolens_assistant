# Time-delay distance

## TL;DR

D_Δt ≡ (1 + z_l) D_l D_s / D_ls. This single combination of angular
diameter distances absorbs all the cosmography of
[[time-delay-cosmography|time-delay measurements]]. For flat ΛCDM, D_Δt
∝ 1/H0 at fixed Ω_m, so a lens-by-lens measurement of D_Δt yields H0.

## What it is

Given a measured Δt between two images and a modelled Fermat potential
difference Δφ from the lens model,

```
D_Δt = c Δt / Δφ
```

Each lens gives an independent D_Δt with ~5–8% precision (Wong 2019,
Birrer 2020).
